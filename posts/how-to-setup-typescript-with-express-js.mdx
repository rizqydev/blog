Setup Typescript with Nodejs and Express.js

```
yarn init
yarn add express helmet dotenv cors mongoose bcrypt jsonwebtoken
yarn add -D @types/bcrypt @types/cors @types/express @types/jsonwebtoken esbuild typescript ts-node-dev eslint prettier
```
